Find any element on the page with JavaScript

Querying the DOM

Every element on a webpage lives inside the document object. JavaScript gives you four built-in methods to find them — each with different behaviour and a different return type.

Try it: Type into any field below. The matching elements in the preview will light up yellow. You can also open your console and run these methods on this page!

Methods
document.getElementById()

Returns a single element by its id. Returns null if not found. IDs must be unique.

Try: main-title  ·  site-nav  ·  page-footer

document.getElementsByTagName()

Returns a live HTMLCollection of every element with that tag name.

Try: p  ·  span  ·  div  ·  a  ·  h2

document.querySelector()

Returns the first element that matches any CSS selector. Returns null if none match.

Try: .card  ·  #main-title  ·  h2

document.querySelectorAll()

Returns a static NodeList of all elements matching a CSS selector.

Try: .tag  ·  .body-text  ·  div.card  ·  p
