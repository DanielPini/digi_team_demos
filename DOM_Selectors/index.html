Find any element on the page with JavaScript

Querying the DOM

Every element on a webpage lives inside the document object. JavaScript gives you four methods to find them — each with different behaviour and a different return type.

Try it: Type a value into any field below. Matching elements light up yellow in the preview. Switch to Page Source to see the highlighted code lines too.

document.getElementById("")
MDN ↗

Returns one element with a matching id attribute. Returns null if nothing matches. IDs must be unique per page.

Try: main-title  ·  site-nav  ·  page-footer

document.getElementsByTagName("")
MDN ↗

Returns a live HTMLCollection of every element with that tag name.

Try: p  ·  span  ·  div  ·  a  ·  h2

document.querySelector("")
MDN ↗

Returns the first element matching any CSS selector. Returns null if nothing matches.

Try: .card  ·  #main-title  ·  h2

document.querySelectorAll("")
MDN ↗

Returns a static NodeList of all elements matching a CSS selector.

Try: .tag  ·  .body-text  ·  div.card  ·  p
